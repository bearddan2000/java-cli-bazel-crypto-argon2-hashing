# java-cli-bazel-crypto-argon2-hashing

## Description
A demo for Argon2 one-way hashing of
a password.

## Tech stack
- java
- bazel

## Docker stack
- ol.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
